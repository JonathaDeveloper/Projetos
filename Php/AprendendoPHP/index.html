Olá, se chegou até aqui espero que esteja bem, é uma honra poder ter você acompanhando esse conteudo.

O Objetivo desse pequeno projeto é demonstrar minha evolução no aprendizado de uma linguagem de programação, escolhi então o PHP para que eu possa está aqui aprendendo, criando e evoluindo no decorrer do tempo e utilizando o livro "Aprendendo PHP por David Sklar". Vou está iniciando a partir do capítulo 2 do livro!

Alem disso quero deixar exemplos aqui nesse projeto, o mais claro e simples possivel para que se torne um bom ponto de referencia, como uma especie de "biblioteca PHP", para que eu e outras pessoas possam usar esse conteudo, podendo então ajudar no dia a dia com dicas e exemplos, criei a estrutura simploria de uma pagina web apenas para ajudar na navegação dos conteudos que serão criados.

Espero está constantimente atualizando esse conteudo até que o mesmo esteja bem objetivo e claro, para que eu possa passar para o proximo nivel,] onde vou está criando outros projetos!

Configuração do meu ambiente:
Servidor : IIS
Versão: PHP 7.4

Descidi não está utilizando o Apache por uma questão de aprendizado e por ter o IIS com um otimo recurso para sites (nativo do Windows [pois é meu S.O atual precisei apenas adicionar o recurso]), não tenho problema em está utilizando um servidor Apache mas acrédito que será melhor quando eu passar para o S.O de alguma distro Linux.
